# REIRIE WordPress Theme

2人組アイドルユニット **REIRIE（レイリエ）** オフィシャルサイト用 WordPress テーマです。

---

## 🎀 特徴

- **動画ヒーロービジョン** + **キラキラパーティクル**（マウス追従）
- **REIRIE 設定ダッシュボード**（管理画面トップに専用メニュー、すべての編集場所への導線を集約）
- **カスタム投稿タイプ**（News / Schedule / Discography / Movie / Member / Goods）で誰でも管理画面から更新可能
- **テーマカスタマイザー**でヒーロー動画/画像・CTAボタン・SNSリンク・お問い合わせカードを変更
- **オリジナルお問い合わせフォーム**（プラグイン不要・AJAX送信・自動返信メール対応）
- **ディスコグラフィ**：購入リンク・Apple Music / Spotify / YouTube Music / まとめリンクのボタンを各作品ごとに設定可能
- **メンバー（アーティスト情報）**：基本情報 / プロフィール詳細 / SNSがタブで分割された見やすい入力UI
- **ACF対応**（無くても標準メタボックスで入力可能）
- 完全レスポンシブ（PC / タブレット / スマホ）
- カスタムカーソル / スクロールリビール演出付き

## 🚀 セットアップ手順

### 1. テーマをインストール

1. WordPress管理画面 → **外観 → テーマ → 新規追加 → テーマのアップロード**
2. `reirie.zip` を選択してアップロード
3. **有効化**

### 2. （推奨）ACFプラグインをインストール

1. **プラグイン → 新規追加** で「Advanced Custom Fields」を検索
2. インストール → 有効化
   - テーマ内の `acf-json/` フォルダから自動的にフィールド定義が読み込まれます
   - 未導入でも標準のメタボックスで入力可能です

### 3. パーマリンク設定

**設定 → パーマリンク** で「**投稿名**」を選択して保存
（カスタム投稿タイプのURLが正しく動くようにするため必須）

### 4. フロントページの設定

1. **固定ページ → 新規追加** で「Home」というタイトルのページを作成（中身は空でOK）
2. **設定 → 表示設定** で：
   - 「ホームページの表示」→ **固定ページ**
   - 「ホームページ」→ **Home**

### 5. ヒーロー動画とSNSリンクを設定

管理画面の **「REIRIE 設定」メニュー**（左サイドバー上部のハートアイコン）からダッシュボードを開くと、各編集場所へのリンクが一覧で表示されます。

または **外観 → カスタマイズ** で以下のセクションを直接編集：

- **REIRIE：ヒーロービジョン**
  - 背景の種類（動画 / 画像）
  - ヒーロー背景動画（MP4）
  - ヒーロー背景画像（動画のポスターとしても使用）
  - オーバーレイの濃さ（0〜80%）
  - サブタイトル / メインタイトル / 日本語タイトル / キャッチコピー
  - **メインボタン・サブボタン**（CTAボタン2つ／ラベルとリンク先URLを設定可能、空欄なら非表示）
  - スライドテロップ（ON/OFF切り替え可能）
- **REIRIE：SNSリンク** … X / Instagram / TikTok / YouTube のURL
- **REIRIE：お問い合わせカード** … FANCLUB / PRESS / FAN MAIL の内容
- **REIRIE：お問い合わせフォーム** … 通知先メール・説明文・プライバシーポリシーURL
- **REIRIE：フッター** … コピーライト表記等

### 6. ロゴ設定（任意）

**外観 → カスタマイズ → サイト基本情報 → ロゴを選択** から差し替え可能。
未設定の場合はテーマ同梱の `assets/img/logo.png` が使われます。

### 7. お問い合わせフォーム設定

1. **固定ページ → 新規追加** で「Contact」というページを作成
2. 右サイドバー「ページ属性 → テンプレート」で **REIRIE お問い合わせフォーム** を選択
3. パーマリンク（スラッグ）を `contact` に設定して公開
4. **外観 → カスタマイズ → REIRIE：お問い合わせフォーム** で以下を設定：
   - 通知先メールアドレス（未設定時はサイトの管理者メールに送信）
   - フォーム上部の説明文
   - プライバシーポリシーページのURL
5. トップページの Contact セクションの3枚カードは、それぞれ以下のフォーム種別へ自動で遷移します：
   - **FANCLUB** → ファンレター（fanmail）
   - **PRESS** → 取材依頼（press）
   - **FAN MAIL** → 出演依頼（casting）

## 📝 コンテンツの登録方法

管理画面の左メニューから各カスタム投稿タイプを選んで登録します。

### 📢 News（お知らせ）

| 項目 | 説明 |
|---|---|
| タイトル | お知らせの見出し |
| カテゴリー | RELEASE / LIVE / MEDIA / GOODS / FANCLUB など自由に作成 |
| 掲載日 | 表示する日付（未入力時は投稿日） |
| 外部リンクURL | 外部サイトに飛ばしたい場合のみ入力 |

### 📅 Schedule（スケジュール）

| 項目 | 説明 |
|---|---|
| タイトル | イベント名 |
| カテゴリー | RELEASE EVENT / LIVE / MEDIA / ONE-MAN など |
| 開催日 | **必須**（カレンダーに表示される日付） |
| 会場 | 例: 渋谷 TSUTAYA O-WEST |
| 開場/開演 | 例: OPEN 18:00 / START 19:00 |
| ハイライト表示 | ワンマンライブ等を目立たせる ✨ |

### 💿 Discography（作品）

| 項目 | 説明 |
|---|---|
| タイトル | 作品名 |
| アイキャッチ画像 | ジャケット画像（正方形推奨 800×800） |
| カテゴリー | 例: 2nd Single / Debut Mini Album |
| リリース日 | 発売日 |
| 価格（任意） | 例: ¥1,500（税込） |
| トラックリスト | 1行に1曲（例: `1. Twinkle Heart`） |
| NEWバッジ表示 | 最新作にチェック |
| 🛒 購入ページURL | CDショップ・公式ストア |
| 🍎 Apple Music | 配信URL |
| 🟢 Spotify | 配信URL |
| ▶ YouTube Music | 配信URL |
| 🔗 まとめリンク | linkco / Linkfire等のURL |

> 入力したリンクのみカード下部にボタン表示されます。

### 🎬 Movie（動画）

| 項目 | 説明 |
|---|---|
| タイトル | 動画タイトル |
| アイキャッチ画像 | サムネイル（16:9 推奨 800×450） |
| 動画URL | YouTube・Vimeo等のURL |

### 👯 Member（メンバー / アーティスト情報）

メンバー編集画面は **「基本情報」「プロフィール詳細」「SNS」** の3タブ構成で、見やすく整理されています。

#### 基本情報タブ
| 項目 | 説明 |
|---|---|
| タイトル | メンバー名（英字、例: Rei） |
| アイキャッチ画像 | プロフィール写真（縦長 4:5 推奨 800×1000） |
| 名前（カナ） | 例: レイ |
| キャッチフレーズ（任意） | 例: 太陽みたいな笑顔担当 |
| メンバーカラー名 | 例: PINK / SKY BLUE / LAVENDER |
| カラークラス | ピンク / スカイブルー / パープル / イエロー / グリーン から選択 |
| フォト背景クラス | Reiスタイル / Rieスタイル / パープル系 から選択 |
| イニシャル | 写真未設定時に表示される一文字 |

#### プロフィール詳細タブ
| 項目 | 説明 |
|---|---|
| 誕生日 / 血液型 / 出身地 | 基本情報 |
| 身長（任意） | 例: 158cm |
| 趣味 / チャームポイント | プロフィール |
| スペシャルスキル（任意） | 例: クラシックバレエ |
| ファンへのメッセージ | 吹き出しに表示される文章 |

#### SNSタブ
| 項目 | 説明 |
|---|---|
| X / Instagram / TikTok / ブログ URL | 入力したものだけがアイコン表示されます |

> **並び順**：右サイドバーの「ページ属性 → 順序」の数字で調整（小さい数字が左）

### 🎬 Movie（動画）

| 項目 | 説明 |
|---|---|
| タイトル | 動画タイトル |
| アイキャッチ画像 | サムネイル（16:9 推奨 800×450） |
| 動画URL | YouTube・Vimeo等のURL |
| 公開日 | 動画の公開日 |
| ラベル | 例: MV / Lyric Video / Behind the Scenes |

### 🛍️ Goods（グッズ）

| 項目 | 説明 |
|---|---|
| タイトル | 商品名 |
| アイキャッチ画像 | 商品画像 |
| 価格表示 | 例: ¥3,800 |

### ✉️ お問い合わせメッセージ

フォームから送信されたメッセージは、管理画面の **お問い合わせ** メニューに自動保存されます。

| 項目 | 説明 |
|---|---|
| 件名 | 送信者が入力した件名 |
| 種別 | ファンレター / 取材依頼 / 出演依頼 / その他 |
| 送信者 | お名前 |
| メール | 返信先メールアドレス |
| 受信日時 | 送信タイムスタンプ |

- 各メッセージを開くと、本文・電話番号・所属・送信元IP等の詳細が表示されます
- 同時に、設定したメールアドレスへ通知メールと、送信者への自動返信メールが配信されます
- スパム対策：ハニーポット・送信時間チェック・WP nonceによる多層ガード

## 🎨 デザインのカスタマイズ

色やフォントを変更したい場合は `assets/css/main.css` の上部にある CSS 変数を編集してください：

```css
:root {
  --pink: #ff7eb6;          /* メインピンク */
  --pink-deep: #e94c8a;     /* アクセントピンク */
  --blue: #8ad4ff;          /* メンバーカラー(青) */
  --purple: #b07aff;        /* アクセント紫 */
  --grad-girl: linear-gradient(135deg, #ff7eb6 0%, #b07aff 50%, #8ad4ff 100%);
}
```

パーティクルの色・量を変更したい場合は `assets/js/particles.js` を編集してください。

## ❓ よくある質問

**Q. ヒーロー動画を変更したい**  
A. **外観 → カスタマイズ → REIRIE：ヒーロービジョン → ヒーロー背景動画** からアップロードできます。MP4形式・10〜15秒・5MB以下推奨。

**Q. メンバーが3人以上に増えた**  
A. Member投稿を追加してください。3人以上になる場合は `assets/css/main.css` の `.member__grid` の `grid-template-columns` を `1fr 1fr 1fr` に変更すると3カラム表示になります。

**Q. パーマリンクが 404 になる**  
A. **設定 → パーマリンク** ページを開いて何もせず「保存」を押してください。リライトルールが再生成されます。

**Q. パーティクルがちょっと多い・少ない**  
A. `assets/js/particles.js` の `ambientCount`（環境光の数）と `count`（マウス追従の生成数）を調整してください。
