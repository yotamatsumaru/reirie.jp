# REIRIE テーマ かんたんセットアップガイド

このガイドは、WordPressをはじめて触る方でも迷わずサイトを公開できるよう、画像なしの最短手順でまとめたものです。

---

## ステップ 1：テーマをアップロードする

1. WordPress管理画面にログイン
2. 左メニュー「**外観**」→「**テーマ**」をクリック
3. 上部の「**新規追加**」ボタン → 「**テーマのアップロード**」
4. `reirie.zip` を選択 → 「**今すぐインストール**」
5. 「**有効化**」をクリック

✅ これでテーマが適用されました。

> 💡 有効化後、左サイドバー上部に **🎀 REIRIE 設定** という専用メニューが追加されます。
> ここから各編集ページへワンクリックでアクセスできるので、迷ったらまずダッシュボードを開いてください。

---

## ステップ 2：パーマリンクを設定する（重要）

「**設定**」→「**パーマリンク**」を開き、

- 「**投稿名**」を選択
- 「**変更を保存**」

を押すだけでOKです。これをしないと News や Schedule の個別ページが 404 になります。

---

## ステップ 3：トップページを「Home」にする

1. 「**固定ページ**」→「**新規追加**」
2. タイトルに「**Home**」と入力（中身は空のままでOK）
3. 「**公開**」
4. 「**設定**」→「**表示設定**」
5. 「**ホームページの表示**」を「**固定ページ**」に変更
6. 「**ホームページ**」で「**Home**」を選択
7. 「**変更を保存**」

✅ サイトトップに REIRIE のデザインが表示されます。

---

## ステップ 4：ACFプラグインを入れる（強く推奨）

1. 「**プラグイン**」→「**新規追加**」
2. 検索バーに「**Advanced Custom Fields**」と入力
3. 一番上に出てくる **Advanced Custom Fields** を「**今すぐインストール**」
4. 「**有効化**」

✅ News や Schedule の編集画面に、見やすい入力欄が出るようになります。

> ※ 入れなくてもサイトは動作しますが、入れた方が圧倒的に楽です。

---

## ステップ 5：SNSリンクとヒーロー動画を設定

「**外観**」→「**カスタマイズ**」を開きます。

### REIRIE：ヒーロービジョン

- **ヒーロー背景動画**：「ファイルを選択」ボタンから動画をアップロード（MP4形式・10〜15秒推奨）
- **メインタイトル / サブタイトル / キャッチコピー** を編集
- **スライドテロップ**：画面下部を流れるテキストを編集

### REIRIE：SNSリンク

- X (Twitter) / Instagram / TikTok / YouTube のURLを入力

### REIRIE：お問い合わせカード

- FANCLUB / PRESS / FAN MAIL の3枚のカード文言（ラベル・タイトル・説明）を編集
- リンク先は自動的に下記のお問い合わせフォーム（種別ごと）に飛びます

### REIRIE：お問い合わせフォーム

- **通知先メールアドレス**：フォーム送信時に通知が届く先（未入力なら管理者メール）
- **フォーム上部の説明文**：お問い合わせページ冒頭に表示される文章
- **プライバシーポリシーURL**：同意チェックボックスからリンクされます

設定が終わったら、上部の「**公開**」をクリック。

---

## ステップ 6：お問い合わせページを作る

1. 「**固定ページ**」→「**新規追加**」
2. タイトルに「**Contact**」と入力
3. 右サイドバーの「**ページ属性 → テンプレート**」で **REIRIE お問い合わせフォーム** を選択
4. 右サイドバーの「**URL（パーマリンク）**」を `contact` に設定
5. 「**公開**」

✅ これでトップページの Contact セクションの3枚カードが、それぞれ専用フォームへ自動でリンクされます：

| カード | フォーム種別 | 遷移先 |
|---|---|---|
| **FANCLUB** | ファンレター | `/contact/?type=fanmail` |
| **PRESS** | 取材依頼 | `/contact/?type=press` |
| **FAN MAIL** | 出演依頼 | `/contact/?type=casting` |

> 送信されたメッセージは、左メニュー「**お問い合わせ**」から閲覧できます。
> 同時に通知メールが届き、送信者にも自動返信メールが配信されます。

---

## ステップ 7：コンテンツを登録する

左メニューに以下のメニューが追加されています：

| メニュー | 用途 |
|---|---|
| **News** | お知らせ・告知 |
| **Schedule** | ライブ・イベント情報 |
| **Discography** | シングル・アルバム |
| **Movie** | MV・動画 |
| **Member** | メンバープロフィール（最初に登録するのがおすすめ） |
| **Goods** | 物販グッズ |

### 最初に登録するもの（推奨順）

1. **Member** … REIRIE のお二人を登録（順序を `0` と `1` にすると左がRei、右がRieになります）
2. **News** … 最新の告知をいくつか
3. **Schedule** … 直近のイベント
4. **Discography** … リリース済みの作品
5. **Movie** … MVのYouTube URL
6. **Goods** … 販売中のグッズ

各投稿の編集画面下に「○○詳細」の入力欄があるので、そこに会場名や価格等を入力します。

---

## ステップ 8：完成！

サイトURL（`https://あなたのドメイン/`）を表示して、デザインが反映されているかチェックしましょう 🎀

---

## 🆘 困ったとき

| 症状 | 対処 |
|---|---|
| News の個別ページが「ページが見つかりません」になる | 「設定 → パーマリンク」を開いて何もせず「変更を保存」 |
| ヒーロー動画が表示されない | カスタマイザーで動画を再アップロード／MP4形式かチェック |
| パーティクルが見えない | ブラウザのキャッシュをクリア（Ctrl+Shift+R） |
| メンバーの並び順を変えたい | Member編集画面右の「ページ属性」→「順序」の数字を変更 |
| カテゴリーが表示されない | News/Scheduleの編集画面でカテゴリーを必ず1つ選択 |

---

ご不明な点はテーマ制作担当までお問い合わせください 🌸
